# React App from Scratch
Author: Tien Le <lehuutien1996@gmail.com>

## Prequisites

- Node.JS with NPM
- Installed Yarn

## Setup Project

### 1. Installing Yarn via NPM

```
npm install -g yarn
```

### 2. Install dependencies via Yarn

```
yarn install
```

### 3. Starting Project

```
yarn start
```
